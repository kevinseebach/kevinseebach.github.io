DoliKingAvis

Module de liaison entre Dolibarr et KingAvis

Ce module est déstiné à être utilisé dans le CRM/ERP Dolibarr afin d'envoyer les demandes d'avis client sur la plateforme KingAvis directement depuis les factures Dolibarr.
L'envoi des demandes peut être automatiser à la validation des factures ou déclencher à la main par l'utilisateur depuis la fiche de la facture. Le mode de fonctionnement est défini par l'utilisateur dans la configuration du module.
